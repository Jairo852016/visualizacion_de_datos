Metodología

El Information Coefficient (IC) mide la correlación de Spearman entre el valor de cada señal y el retorno forward a 21 días. Un IC positivo indica que la señal predice correctamente la dirección del retorno. Para este conjunto de datos, los valores oscilan entre −0.20 y +0.20; cualquier IC sostenido por encima de 0.05 se considera económicamente significativo.

Los datos provienen de yfinance (precios de cierre ajustados, 2015–2024). Las señales se calculan sin look-ahead bias: cada valor usa únicamente información disponible en esa fecha.

Explorar el Análisis

El análisis sigue un flujo de dos etapas: primero se construye el dataset limpio con las señales técnicas, luego se responden las preguntas analíticas con visualizaciones interactivas. Cada etapa es independiente — los datos procesados se guardan en signals.parquet para que las visualizaciones carguen al instante.
